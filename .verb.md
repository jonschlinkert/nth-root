## Usage

```js
var nthRoot = require('{%= name %}');
```

Returns the square root of the given number by default:

```js
nthRoot(16),
//=> '4'
nthRoot(-16), 
//=> '-4'
```

Pass a degree as the second param. e.g. to get cube root:

```js
nthRoot(-27, 3), 
//=> '-3'
nthRoot(27, 3),
//=> '3'
nthRoot(-8, 3), 
//=> '-2'
nthRoot(8, 3),
//=> '2'
```
